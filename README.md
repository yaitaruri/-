## 📖 遊戲簡介 (Introduction)

採用Cyberpunk風格的文字RPG，玩家將扮演潛入深網的頂尖駭客，透過神經連結裝置突破層層防火牆。從街頭的簡易設備，一路駭入商業公司、網路監察局，最終挑戰失控AI與傳奇改造人。遊戲介面特別模擬了復古駭客終端機的視覺效果，並加入打字機文字特效，營造身歷其境的駭客體驗。

## 🏙️ 甚麼是 Cyberpunk

賽博龐克是人類對未來幻想的虛擬世界，但並非美好的烏托邦。在這個世界裡，科技極度發達，人類可以隨意改造身體或上傳意識，但生活品質卻反倒低落。巨大的企業掌控了世界，貧富差距極大。像是知名的遊戲《Cyberpunk 2077》與動畫《賽博龐克：邊緣行者》，就是以這個架構為背景的代表作。

## ✨ 遊戲特色 (Features)

* **仿復古駭客終端機設計**：營造身歷其境的駭客體驗。
* **5個階段共5隻Boss15隻小怪**：豐富的敵人配置與關卡進程。
* **每隻敵人都有不同的特色技能**：從燃燒、護盾到即死攻擊，考驗策略應對。
* **多種技能選擇**：裝配最適合自己的技能組（如：超頻運算、記憶體洩漏）。
* **動態的市集價格**：隨著玩家升級物品會越來越貴，考驗資源管理。

## 👾 敵人資訊 (Database)

⚠️ **警告：偵測到大量敵意訊號**

### Level 1: 街頭公網 (★)
* **[BOSS] 非法挖礦主機**：每回合造成額外燒傷。
* **老舊的家用路由器**：最基礎的練習目標，防護薄弱。
* **路邊監視器**：若戰鬥拖太久，攻擊力會加倍。
* **腳本小子**：攻擊不穩定，傷害浮動大。

### Level 2: 商業公司 (★★)
* **[BOSS] 機房守衛機器人**：殘血時會緊急修復自身。
* **公司值班電腦**：裝有基礎防護，適合練手。
* **防毒軟體 v2.0**：擁有物理減傷護盾，建議用病毒技能攻擊。
* **加密資料庫**：防禦系統運作中，釋放微量電流（通常帶有高防禦或反傷特性）。

### Level 3: 荒坂塔 (★★★)
* **[BOSS] 神機錯亂者**：血量越低，攻擊力越狂暴。
* **荒坂鎮暴機甲**：重型裝甲，防禦力極高。
* **智慧型砲塔**：智慧鎖定，攻擊無法閃避。
* **軍用無人機**：高速迴旋，有機率發動二連擊。

### Level 4: 網路監察 (★★★★)
* **[BOSS] 鎮暴部隊總長**：施放戰術煙霧，閃避率極高。
* **網監攔截程式**：開場封鎖玩家技能。
* **追蹤衛星訊號**：特殊倒數機制，回合結束後進行軌道轟炸。
* **黑牆閘門**：恐懼靈氣，攻擊會大幅扣除玩家的 HP 上限。

### Level 5: 深網 (★★★★★)
* **[FINAL BOSS] 亞當碎骨**：傳奇改造人，擁有全能戰力。
* **失控 AI**：行為混沌，攻擊模式隨機切換。
* **毀滅性病毒**：死亡時會自爆，造成真實傷害。
* **舊時代的數據幽靈**：攻擊會吸取玩家 RAM（魔力/行動點）。

## 🚀 快速開始 (Quick Start)

本遊戲為 C++ 開發的 Windows 應用程式，內建編譯與執行腳本。

### 下載與安裝
1. 下載此專案代碼 (Clone or Download ZIP)。
2. 確認電腦已安裝 `MinGW (g++)` 編譯器。
3. 直接雙擊資料夾中的 **`start.bat`** 即可自動編譯並開始遊戲。

### 手動編譯 (Manual Compilation)
如果您想手動編譯，請使用以下指令（需支援 C++17 與 UTF-8 編碼）：

```bash
g++ -std=c++17 -finput-charset=UTF-8 -fexec-charset=UTF-8 main.cpp -o main.exe
./main.exe
